# Archived earlier paper material

These files document an **earlier** stage of the project and do **not** apply to
the release paper (`paper/prop51.tex`). They are kept only for historical
reference.

- `prop51_tenth_revision_note.tex` — the tenth-revision technical research note,
  the pre-formalization human draft. The release paper supersedes it.
- `ERRATA_tenth_revision.md` — errata found, during formalization, against that
  tenth-revision note (an `N`-edge scan correction and a clarifying remark in the
  positive-part section). These issues were addressed in the formalization and in
  the release paper, so the errata are of historical interest only.
